# Como Precision Website

Static Astro site for [szcomo.com](https://szcomo.com/).

The site presents Como Precision as the company hub for:

- Industrial 3D printing
- Metal 3D printing powders
- Titanium additive manufacturing
- Copper additive manufacturing
- Precision ceramic machining

Project inquiries go to `info@szcomo.com`.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
